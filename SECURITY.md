# Security Policy

This repository hosts the **svg-engine website**. The policy below covers the
website; vulnerabilities in the svg-engine library can be reported through the
same channel.

## Reporting a vulnerability

**Do not open a public issue for security reports.**

Email **svgengine@mosaicoo.tech** with:

- a clear description of the issue;
- steps to reproduce (or a proof of concept);
- the potential impact;
- the affected URL, page or commit.

We will acknowledge your report as soon as possible — typically within a few
business days — and keep you informed about the fix. Please give us reasonable
time to address the issue before any public disclosure.

## Scope

In scope:

- this website (`svgengine-site`) and its build/deploy pipeline.

Out of scope:

- issues that require a compromised user device or browser;
- best-practice suggestions without a concrete, demonstrable impact.

Thank you for helping keep svg-engine and its users safe.
