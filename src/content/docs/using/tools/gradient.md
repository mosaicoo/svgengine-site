**Shortcut: `G`.** The Gradient tool focuses the selected shape and opens the
gradient panel so you can edit its gradient.

## How to use it

With a shape selected, activate the Gradient tool. It focuses the shape and
brings up the gradient panel. The gradient's **stops** and **axis** are edited
directly on the canvas through the gradient overlay (draggable stop handles), and
in the gradient panel.

This tool has no options of its own — the editing happens in the overlay and
panel.

## Tips & interactions

- The gradient overlay and editor also activate from the selection, so you can
  refine a gradient without keeping this tool active.
