**Shortcut: `Shift`+`O`.** The Page tool manages **pages** (artboards) — it
follows the Illustrator Artboard Tool pattern.

## How to use it

With the Page tool active, the page chrome appears on the canvas. **Drag** an
empty area to create a new page; **drag a page's handles or brackets** to move or
resize it. The page chrome is only interactive while this tool is active.

Pages can also be created, renamed and reordered from the **Pages panel**.

## Tips & interactions

- Each page is an artboard; the **active page** is where newly drawn objects are
  added.
- Use pages to keep multiple variations or screens in one document.
