---
title: Eyedropper tool
description: Sample a style from one object and apply it to the selection.
---

**Shortcut: `I`.** The Eyedropper samples an object's style (fill and/or stroke)
and applies it to your current selection.

## How to use it

Click an object to sample its style. With **Auto-apply** on, the sampled style is
applied to the current selection immediately.

## Tool Options

| Option | Values | What it does |
| --- | --- | --- |
| Target | Fill / Stroke / Both | Which part of the style to sample |
| Auto-apply | on / off | Off only samples (no change applied) |
| Reset | — | Restore the default options |

## Tips & interactions

- Use **Both** to copy the full appearance; **Fill** or **Stroke** to copy just
  one.
