title: Knife tool
description: Cut and split paths.
---

**Shortcut: `C`.** The Knife cuts across a path and splits it where you cut.

## How to use it

Drag across a path to slice it; the path is split at the cut. With **Snap to
nodes** on, the cut snaps to nearby existing anchors.

## Tool Options

| Option | Range | What it does |
| --- | --- | --- |
| Snap to nodes | on / off | Snap the cut to existing anchor points |
| Tolerance | 2–100 | Snap distance, in pixels |
| Reset | — | Restore the default options |

## Tips & interactions

- After cutting, use [Select](/svgengine-site/using/tools/select/) to move the
  resulting pieces apart.
