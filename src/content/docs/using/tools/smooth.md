title: Smooth tool
description: Smooth and simplify paths.
---

**Shortcut: `S`.** The Smooth tool reduces a path's anchor points and smooths its
curves.

## How to use it

Apply it to a path to simplify it — fewer anchors, smoother shape. The amount of
simplification is controlled by **Tolerance**.

## Tool Options

| Option | Range | What it does |
| --- | --- | --- |
| Tolerance | 0.1–20 | Simplification strength (RDP epsilon, in doc units) — higher removes more points |
| Reset | — | Restore the default options |

## Tips & interactions

- Raise **Tolerance** for stronger simplification; lower it to keep more detail.
- Great for cleaning up [Pencil](/svgengine-site/using/tools/pencil/) strokes.
