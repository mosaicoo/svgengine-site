**Shortcut: `O`.** The Symbol Sprayer scatters instances of a symbol as you drag.

## How to use it

Pick the symbol to spray, then **drag** on the canvas to place instances along
the path of your cursor. A **live preview** shows the drops while you drag, and
the whole spray commits as a **single undo** on release.

## Tool Options

| Option | Range | What it does |
| --- | --- | --- |
| Symbol | — | The symbol to spray |
| Size | 4–200 | Base size of each instance |
| Spacing | 2–200 | Distance between drops, in pixels |
| Jitter | from 0 | Random ± scale variation per drop |
| Reset | — | Restore the default options |

## Tips & interactions

- Raise **Jitter** for a more natural, scattered look.
- Symbols come from the symbol library; instances stay linked to their master.
