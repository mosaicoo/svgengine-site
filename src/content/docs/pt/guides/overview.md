O **svg-engine** é uma library Angular para **renderizar, manipular e otimizar
SVG**, com uma camada de editor visual completa, desenhada para ser **embutida
em aplicações de terceiros**.

## Posicionamento

O produto principal é o **engine headless** — o conjunto de entry points sem UI
obrigatória. A UI profissional pronta é uma **camada de conveniência opt-in**,
substituível pelo consumidor.

## Para quem é

- Desenvolvedores Angular que precisam de um editor/render de SVG embutido.
- Produtos SaaS / white-label que querem edição vetorial sem reconstruir o core.
- Times com requisitos de acessibilidade e privacidade.
- Colaboradores open source (autores de plugins).
