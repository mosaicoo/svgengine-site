Tutoriais práticos que combinam as [ferramentas](/svgengine-site/pt/using/tools/)
e os [conceitos](/svgengine-site/pt/using/concepts/). Experimente na
[demo ao vivo](/svgengine-site/pt/demo/).

## 1. Desenhe sua primeira forma

1. Pressione `R` (ou clique no **Rectangle** na paleta de ferramentas).
2. **Arraste** no canvas para desenhar um retângulo. Segure **Shift** para um
   quadrado, ou **Alt** para desenhar a partir do centro.
3. Pressione `V` para ir ao **Select** e clique na sua forma.
4. No **Inspector**, defina uma cor de **fill**. Pronto — seu primeiro objeto.

## 2. Faça um retângulo arredondado

1. Pressione `R` de novo.
2. Na barra de **Tool Options**, aumente o **Corner radius**.
3. Arraste um novo retângulo — os cantos ficam arredondados.

## 3. Edite os pontos de um path

1. Desenhe uma forma, selecione-a e converta-a em **path**.
2. Pressione `A` para o **Direct Select** — os anchor points aparecem.
3. **Arraste um anchor** para movê-lo; **duplo-clique num anchor** para alternar
   o tipo (cusp ↔ smooth ↔ symmetric); **duplo-clique num segmento** para
   adicionar um ponto.

## 4. Combine duas formas

1. Desenhe duas formas sobrepostas.
2. Pressione `V` e **Shift-clique** nas duas para selecioná-las.
3. Aplique uma operação do **Pathfinder** — tente **Union** para unir, ou
   **Subtract** para subtrair uma da outra.

## 5. Desenhe um traço livre e limpe-o

1. Pressione `P` para o **Pencil** e desenhe uma linha à mão livre.
2. Pressione `S` para o **Smooth** e aplique para simplificar a linha.
3. Vá para o **Direct Select** (`A`) para ajustes finais.
