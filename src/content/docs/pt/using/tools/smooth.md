**Atalho: `S`.** A ferramenta Smooth reduz os anchor points de um path e suaviza
suas curvas.

## Como usar

Aplique a um path para simplificá-lo — menos anchors, forma mais suave. A
quantidade de simplificação é controlada pela **Tolerance**.

## Tool Options

| Opção | Faixa | O que faz |
| --- | --- | --- |
| Tolerance | 0.1–20 | Força da simplificação (epsilon RDP, em unidades de doc) — maior remove mais pontos |
| Reset | — | Restaura as opções padrão |

## Dicas & interações

- Aumente a **Tolerance** para simplificação mais forte; reduza para manter mais
  detalhe.
- Ótima para limpar traços do [Pencil](/svgengine-site/pt/using/tools/pencil/).
