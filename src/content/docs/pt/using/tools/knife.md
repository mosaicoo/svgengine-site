**Atalho: `C`.** O Knife corta um path e o divide onde você corta.

## Como usar

Arraste sobre um path para fatiá-lo; o path é dividido no corte. Com **Snap to
nodes** ligado, o corte gruda em anchors existentes próximos.

## Tool Options

| Opção | Faixa | O que faz |
| --- | --- | --- |
| Snap to nodes | on / off | Gruda o corte em anchor points existentes |
| Tolerance | 2–100 | Distância de snap, em pixels |
| Reset | — | Restaura as opções padrão |

## Dicas & interações

- Depois de cortar, use a [Select](/svgengine-site/pt/using/tools/select/) para
  afastar as partes resultantes.

Veja a [visão geral das ferramentas](/svgengine-site/pt/using/tools/) e os
[atalhos de teclado](/svgengine-site/pt/using/keyboard-shortcuts/).
