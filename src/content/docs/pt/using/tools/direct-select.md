title: Ferramenta Direct Select
description: Editar os anchor points e handles individuais de um path.
---

**Atalho: `A`.** A Direct Select edita um path no nível dos anchors — movendo
pontos, remodelando curvas e mudando o tipo de anchor.

## Como usar

Com um path ativo, a Direct Select revela seus anchor points e handles:

- **Arraste um anchor** para movê-lo; arraste os **handles** para remodelar as
  curvas adjacentes.
- **Duplo-clique num segmento** para adicionar um anchor onde você clicar.
- **Alt-clique num segmento** para adicionar um anchor no ponto médio dele.
- **Duplo-clique num anchor** para alternar o tipo: **cusp ↔ smooth ↔
  symmetric**.
- Selecione anchors e pressione **Delete** para removê-los; as **setas** movem os
  anchors selecionados.

Os três tipos de anchor são explicados nos
[conceitos de edição vetorial](/svgengine-site/pt/using/concepts/).

## Tool Options

A barra de Tool Options mostra um lembrete dessas interações com teclas
modificadoras enquanto a ferramenta está ativa.

## Dicas & interações

- Para editar anchors de uma forma (retângulo, elipse, polígono…), converta-a em
  path primeiro.
- Volte para a [Select](/svgengine-site/pt/using/tools/select/) (`V`) para mover
  ou transformar o objeto inteiro.
