**Atalho: `Shift`+`O`.** A ferramenta Page gerencia **páginas** (artboards) —
segue o padrão Artboard Tool do Illustrator.

## Como usar

Com a ferramenta Page ativa, o chrome de página aparece no canvas. **Arraste**
uma área vazia para criar uma nova página; **arraste os handles ou brackets de
uma página** para movê-la ou redimensioná-la. O chrome de página só é interativo
enquanto esta ferramenta está ativa.

As páginas também podem ser criadas, renomeadas e reordenadas pelo **Pages
panel**.

## Dicas & interações

- Cada página é um artboard; a **página ativa** é onde os novos objetos
  desenhados são adicionados.
- Use páginas para manter várias variações ou telas em um único documento.
