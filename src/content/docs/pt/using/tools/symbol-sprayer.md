**Atalho: `O`.** O Symbol Sprayer espalha instâncias de um símbolo conforme você
arrasta.

## Como usar

Escolha o símbolo a espalhar e **arraste** no canvas para colocar instâncias ao
longo do caminho do cursor. Um **preview ao vivo** mostra as gotas durante o
arrasto, e todo o spray é confirmado como um **único undo** ao soltar.

## Tool Options

| Opção | Faixa | O que faz |
| --- | --- | --- |
| Symbol | — | O símbolo a espalhar |
| Size | 4–200 | Tamanho base de cada instância |
| Spacing | 2–200 | Distância entre as gotas, em pixels |
| Jitter | a partir de 0 | Variação aleatória ± de escala por gota |
| Reset | — | Restaura as opções padrão |

## Dicas & interações

- Aumente o **Jitter** para um visual mais natural e espalhado.
- Os símbolos vêm da symbol library; as instâncias permanecem ligadas ao master.
