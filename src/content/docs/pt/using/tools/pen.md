**Atalho: `B`.** A ferramenta Pen constrói paths vetoriais ponto a ponto —
cantos com cliques, curvas com click-drags.

## Como usar

- **Clique** para colocar um anchor de **canto** (cusp).
- **Clique e arraste** para colocar um anchor **smooth** com handles simétricos —
  isso curva o path por aquele ponto.
- Uma linha de preview **rubber-band** acompanha o cursor entre os cliques.
- **Clique novamente no primeiro anchor** para **fechar** o path e finalizá-lo.
- **Enter** finaliza o path aberto atual; **Esc** o descarta.

## Tool Options

| Opção | Faixa | O que faz |
| --- | --- | --- |
| Fill | cor / nenhum | Cor de preenchimento do novo path (com toggle de no-fill) |
| Stroke | cor | Cor do traço (contorno) |
| Stroke width | 0.5–100 | Espessura do contorno |
| Rubber band | on / off | Mostra a linha de preview durante o desenho |
| Reset | — | Restaura as opções padrão |

## Dicas & interações

- Misture **cliques** (cantos) e **click-drags** (curvas) para desenhar qualquer
  contorno.
- Refine o resultado depois com a ferramenta
  [Direct Select](/svgengine-site/pt/using/tools/direct-select/).

Veja a [visão geral das ferramentas](/svgengine-site/pt/using/tools/) e os
