**Atalho: `I`.** O Eyedropper captura o estilo de um objeto (fill e/ou stroke) e
o aplica à seleção atual.

## Como usar

Clique num objeto para capturar o estilo dele. Com **Auto-apply** ligado, o
estilo capturado é aplicado imediatamente à seleção atual.

## Tool Options

| Opção | Valores | O que faz |
| --- | --- | --- |
| Target | Fill / Stroke / Both | Qual parte do estilo capturar |
| Auto-apply | on / off | Off apenas captura (sem aplicar) |
| Reset | — | Restaura as opções padrão |

## Dicas & interações

- Use **Both** para copiar a aparência completa; **Fill** ou **Stroke** para
  copiar só uma.
