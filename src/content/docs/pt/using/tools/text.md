**Atalho: `T`.** A ferramenta Text adiciona texto e o edita inline no canvas.

## Como usar

- **Clique** no canvas para inserir um nó de texto e entrar no modo de edição
  inline; aparece um placeholder que você digita por cima.
- **Enter** ou clicar fora (**blur**) confirma o texto. Texto vazio em um
  placeholder novo é removido automaticamente.
- **Esc** cancela a edição.
- **Shift+Enter** insere uma quebra de linha.

Defina **fonte, tamanho, peso e alinhamento nas opções antes de clicar**, para o
novo texto já começar do jeito que você quer.

## Tool Options

| Opção | Faixa | O que faz |
| --- | --- | --- |
| Font | família de fonte | Tipografia do novo texto |
| Size | 6–400 | Tamanho da fonte, em px |
| Weight | peso da fonte | Espessura/negrito do texto |
| Italic | on / off | Estilo itálico |
| Anchor | esquerda / centro / direita | Alinhamento (`text-anchor` start / middle / end) |
| Fill | cor | Cor do texto |
| Reset | — | Restaura as opções padrão |

## Dicas & interações

- O texto suporta runs de rich-text, variable fonts e texto sobre path; configure
  isso pelo Inspector depois de criar o texto.
