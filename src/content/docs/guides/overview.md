**svg-engine** is an Angular library for **rendering, manipulating and
optimizing SVG**, with a full visual editor layer, designed to be **embedded
into third-party applications**.

## Positioning

The core product is the **headless engine** — the set of entry points with no
mandatory UI. The ready-made professional UI is an **opt-in convenience layer**,
replaceable by the consumer.

## Who it is for

- Angular developers who need an embedded SVG editor/renderer.
- SaaS / white-label products that want vector editing without rebuilding the core.
- Teams with accessibility and privacy requirements.
- Open-source contributors (plugin authors).
