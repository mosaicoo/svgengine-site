---
title: Herramienta Direct Select
description: Editar los anchor points y handles individuales de un path.
---

**Atajo: `A`.** Direct Select edita un path a nivel de anchors — moviendo puntos,
remodelando curvas y cambiando el tipo de anchor.

## Cómo usarla

Con un path activo, Direct Select revela sus anchor points y handles:

- **Arrastra un anchor** para moverlo; arrastra los **handles** para remodelar
  las curvas adyacentes.
- **Doble clic en un segmento** para añadir un anchor donde hagas clic.
- **Alt-clic en un segmento** para añadir un anchor en su punto medio.
- **Doble clic en un anchor** para alternar su tipo: **cusp ↔ smooth ↔
  symmetric**.
- Selecciona anchors y pulsa **Delete** para eliminarlos; las **flechas** mueven
  los anchors seleccionados.

Los tres tipos de anchor se explican en los
[conceptos de edición vectorial](/svgengine-site/es/using/concepts/).

## Tool Options

La barra de Tool Options muestra un recordatorio de estas interacciones con
teclas modificadoras mientras la herramienta está activa.

## Consejos e interacciones

- Para editar los anchors de una forma (rectángulo, elipse, polígono…),
  conviértela en path primero.
- Vuelve a [Select](/svgengine-site/es/using/tools/select/) (`V`) para mover o
  transformar el objeto entero.

Mira el [resumen de herramientas](/svgengine-site/es/using/tools/) y los
[atajos de teclado](/svgengine-site/es/using/keyboard-shortcuts/).
