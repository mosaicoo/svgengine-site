title: Herramienta Width
description: Da al trazo de un path un perfil de ancho variable.
---

**Atajo: `W`.** La herramienta Width aplica un perfil de ancho variable al trazo
de un path seleccionado.

## Cómo usarla

Aplícala a un path seleccionado para remodelar su trazo en un contorno con perfil
(más grueso y más fino a lo largo). El cambio se hornea en la geometría y se
puede revertir con undo.

## Tool Options

| Opción | Valores | Qué hace |
| --- | --- | --- |
| Profile | uniform / tapered / calligraphic | La forma del ancho a lo largo del trazo |
| Width | 1–100 | Ancho base, en píxeles |
| Reset | — | Restaura las opciones por defecto |

## Consejos e interacciones

- **Tapered** es grueso en el medio; **calligraphic** va de fino a grueso.
- Está relacionada con la expansión por brush del
  [Pencil](/svgengine-site/es/using/tools/pencil/) — ambas moldean un trazo con
  un perfil de ancho.
