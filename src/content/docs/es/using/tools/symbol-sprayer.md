**Atajo: `O`.** El Symbol Sprayer esparce instancias de un símbolo mientras
arrastras.

## Cómo usarla

Elige el símbolo a rociar y **arrastra** en el canvas para colocar instancias a
lo largo del recorrido del cursor. Una **vista previa en vivo** muestra las gotas
mientras arrastras, y todo el rociado se confirma como un **único undo** al
soltar.

## Tool Options

| Opción | Rango | Qué hace |
| --- | --- | --- |
| Symbol | — | El símbolo a rociar |
| Size | 4–200 | Tamaño base de cada instancia |
| Spacing | 2–200 | Distancia entre gotas, en píxeles |
| Jitter | desde 0 | Variación aleatoria ± de escala por gota |
| Reset | — | Restaura las opciones por defecto |

## Consejos e interacciones

- Sube el **Jitter** para un aspecto más natural y disperso.
- Los símbolos vienen de la symbol library; las instancias siguen vinculadas al
  master.
