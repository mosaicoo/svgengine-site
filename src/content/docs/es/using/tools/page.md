title: Herramienta Page
description: Crea y gestiona páginas / artboards.
---

**Atajo: `Shift`+`O`.** La herramienta Page gestiona **páginas** (artboards) —
sigue el patrón Artboard Tool de Illustrator.

## Cómo usarla

Con la herramienta Page activa, el chrome de página aparece en el canvas.
**Arrastra** un área vacía para crear una nueva página; **arrastra los handles o
brackets de una página** para moverla o redimensionarla. El chrome de página solo
es interactivo mientras esta herramienta está activa.

Las páginas también se pueden crear, renombrar y reordenar desde el **Pages
panel**.

## Consejos e interacciones

- Cada página es un artboard; la **página activa** es donde se añaden los nuevos
  objetos dibujados.
- Usa páginas para mantener varias variaciones o pantallas en un solo documento.

Mira el [resumen de herramientas](/svgengine-site/es/using/tools/) y los
[atajos de teclado](/svgengine-site/es/using/keyboard-shortcuts/).
