**Atajo: `V`.** Select es la herramienta por defecto: selecciona objetos y
ejecuta el comportamiento estándar del canvas — selección, marquee, mover y snap.

## Cómo usarla

- **Haz clic** en un objeto para seleccionarlo; clic en un espacio vacío para
  deseleccionar.
- **Shift-clic** (o **Ctrl-clic**) para añadir o quitar de la selección.
- **Arrastra un área vacía** para dibujar un marquee y seleccionar lo que toque.
- **Arrastra un objeto seleccionado** para moverlo. El snap lo alinea a otros
  objetos y a la cuadrícula mientras arrastras.
- **Arrastra los handles** para escalar; **arrastra el handle de rotación** para
  rotar. El resize y la rotación ocurren alrededor de un **pivot** móvil, y el
  resultado se hornea en geometría real para que los strokes y las esquinas no se
  distorsionen.
- **Doble clic en un grupo** para entrar en él (isolation) y editar su contenido;
  un breadcrumb muestra dónde estás.

## Tool Options

Con objetos seleccionados, la barra de Tool Options ofrece acciones de
organización:

- **Align** — izquierda, centro (horizontal), derecha, arriba, centro
  (vertical), abajo.
- **Distribute** — horizontalmente, verticalmente.
- **Flip** — horizontal, vertical.

## Consejos e interacciones

- Los objetos bloqueados no se pueden seleccionar — desbloquéalos en el panel de
  capas.
- Para editar los puntos individuales de un path en lugar del objeto entero,
  cambia a la herramienta [Direct Select](/svgengine-site/es/using/tools/direct-select/).

Mira el [resumen de herramientas](/svgengine-site/es/using/tools/) y los
[atajos de teclado](/svgengine-site/es/using/keyboard-shortcuts/).
