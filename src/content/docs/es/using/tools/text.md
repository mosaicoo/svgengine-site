**Atajo: `T`.** La herramienta Text añade texto y lo edita en línea en el canvas.

## Cómo usarla

- **Haz clic** en el canvas para insertar un nodo de texto y entrar en modo de
  edición en línea; aparece un placeholder sobre el que escribes.
- **Enter** o hacer clic fuera (**blur**) confirma el texto. El texto vacío en un
  placeholder nuevo se elimina automáticamente.
- **Esc** cancela la edición.
- **Shift+Enter** inserta un salto de línea.

Define la **fuente, el tamaño, el peso y la alineación en las opciones antes de
hacer clic**, para que el nuevo texto empiece como quieres.

## Tool Options

| Opción | Rango | Qué hace |
| --- | --- | --- |
| Font | familia de fuente | Tipografía del nuevo texto |
| Size | 6–400 | Tamaño de fuente, en px |
| Weight | peso de fuente | Grosor/negrita del texto |
| Italic | on / off | Estilo cursiva |
| Anchor | izquierda / centro / derecha | Alineación (`text-anchor` start / middle / end) |
| Fill | color | Color del texto |
| Reset | — | Restaura las opciones por defecto |

## Consejos e interacciones

- El texto admite runs de rich-text, variable fonts y texto sobre path;
  configúralos desde el Inspector tras crear el texto.
