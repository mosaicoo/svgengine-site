**Atajo: `I`.** El Eyedropper captura el estilo de un objeto (fill y/o stroke) y
lo aplica a la selección actual.

## Cómo usarla

Haz clic en un objeto para capturar su estilo. Con **Auto-apply** activado, el
estilo capturado se aplica de inmediato a la selección actual.

## Tool Options

| Opción | Valores | Qué hace |
| --- | --- | --- |
| Target | Fill / Stroke / Both | Qué parte del estilo capturar |
| Auto-apply | on / off | Off solo captura (sin aplicar) |
| Reset | — | Restaura las opciones por defecto |

## Consejos e interacciones

- Usa **Both** para copiar la apariencia completa; **Fill** o **Stroke** para
  copiar solo una.
