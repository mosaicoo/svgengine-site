**Atajo: `S`.** La herramienta Smooth reduce los anchor points de un path y
suaviza sus curvas.

## Cómo usarla

Aplícala a un path para simplificarlo — menos anchors, forma más suave. La
cantidad de simplificación la controla la **Tolerance**.

## Tool Options

| Opción | Rango | Qué hace |
| --- | --- | --- |
| Tolerance | 0.1–20 | Fuerza de simplificación (epsilon RDP, en unidades de doc) — mayor elimina más puntos |
| Reset | — | Restaura las opciones por defecto |

## Consejos e interacciones

- Sube la **Tolerance** para una simplificación más fuerte; bájala para mantener
  más detalle.
- Ideal para limpiar trazos del [Pencil](/svgengine-site/es/using/tools/pencil/).
