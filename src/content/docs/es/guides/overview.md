**svg-engine** es una librería Angular para **renderizar, manipular y optimizar
SVG**, con una capa de editor visual completa, diseñada para ser **embebida en
aplicaciones de terceros**.

## Posicionamiento

El producto principal es el **motor headless** — el conjunto de entry points sin
UI obligatoria. La UI profesional lista para usar es una **capa de conveniencia
opt-in**, reemplazable por el consumidor.

## Para quién es

- Desarrolladores Angular que necesitan un editor/render de SVG embebido.
- Productos SaaS / white-label que quieren edición vectorial sin reconstruir el
  core.
- Equipos con requisitos de accesibilidad y privacidad.
- Colaboradores open source (autores de plugins).
